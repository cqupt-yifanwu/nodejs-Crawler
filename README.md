# nodejs-Crawler
nodejs入门学习,利用nodejs做的一个爬虫
<h4>利用了如下技术</h4>
<ul>
	<li>SuperAgent(处理http)</li>
	<li>eventproxy(处理并发)</li>
	<li>cheerio(将得到的数据转化为可以用jquery处理的格式)</li>
</ul>
